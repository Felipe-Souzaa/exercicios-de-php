 Lista de PHP - Parte 4 Repetição For Ex04

***

Modifique o exercício anterior para que ele imprima todos os números primos desde intervalo.

## Resolução do exercício:

```

<?php
$num = 0;
for ($i = 190; $i >= 130 && $num < 30; $i--) {
    $teste = true;

    if ($i <= 1) {
        $teste = false;
    } else {
        for ($j = 2; $j <= sqrt($i); $j++) {
            if ($i % $j === 0) {
                $teste = false;
                break;
            }
        }
    }

    if ($teste) {
        echo $i . " \n";
        $num++;
    }
}
?>

```
